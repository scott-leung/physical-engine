## Others

### static

默认情况下，`GitHub Pages` 使用 `Jekyll` 来运行托管的网站。由于 `Jekyll` 会丢弃所有以 `_`字符开头的文件，因此，如果你使用 `GitHub pages` 托管你的网站，建议你通过将名为 `.nojekyll` 的空文件添加到 `static` 目录中来禁用 `Jekyll`。
